# Obsidian Sync (유료)

## 개요

Obsidian이 공식 제공하는 클라우드 동기화 서비스. 별도 Git 설정 없이 Obsidian 앱 내에서 완결된다.

## 가격

| 플랜 | 가격 | 용량 | 버전 히스토리 |
|------|------|------|--------------|
| Sync Standard | $4/월 | 1GB | 1년 |
| Sync Plus | $10/월 | 10GB | 1년 + 고급 기능 |

연간 결제 시 약 20% 할인.

## 설정 방법

1. [obsidian.md/sync](https://obsidian.md/sync) 에서 계정 생성
2. Obsidian 앱 → Settings → Sync → Log in
3. Remote vault 생성 또는 기존 vault 연결
4. 두 번째 기기에서 동일 계정 로그인 → 자동 동기화 시작

## 장점

**설정 간편함:** Git 지식 없이 시작 가능. 앱 내에서 5분 내 완료.

**실시간 동기화:** 편집 즉시 다른 기기에 반영 (파일 저장 기준).

**버전 히스토리:** 파일별 최대 1년치 히스토리. 실수로 삭제한 내용 복원 가능.

**선택적 암호화:** 동기화 전 클라이언트 측 암호화 옵션. 민감한 내용이 있다면 활성화.

**공식 지원:** Obsidian 팀의 공식 서비스. 앱 업데이트와 함께 안정성 보장.

## 단점

**유료:** 무료 대안 (Git 기반) 대비 월 비용 발생.

**용량 제한:** 1GB (기본). 이미지, PDF가 많으면 빠르게 소진.

**버전 히스토리 별도 관리:** Git처럼 커밋 메시지·브랜치 관리는 불가.

**Obsidian 종속:** 서비스가 종료되거나 요금이 오르면 이전 필요.

## Git 기반과 비교

| 구분 | Obsidian Sync | Git 기반 |
|------|---------------|---------|
| 비용 | $4-10/월 | 무료 |
| 설정 | 쉬움 (5분) | 보통 (30분) |
| 버전 관리 | 1년 히스토리 | 무제한 (Git) |
| 커밋 메시지 | 없음 | 직접 작성 |
| 충돌 처리 | 자동 | 수동 |
| 플랫폼 | 전 플랫폼 | 전 플랫폼 |
| 팀 협업 | 지원 (Extra 플랜) | Git으로 가능 |

## 권장 대상

- Git을 사용하지 않거나 배우고 싶지 않은 경우
- 설정 없이 즉시 멀티 디바이스 동기화가 필요한 경우
- 월 $4-10의 비용이 부담되지 않는 경우
- 이미지, PDF가 적고 텍스트 위주로 wiki를 운영하는 경우

## 주의

- `personal/` 폴더의 민감한 내용이 클라우드에 저장됨. 암호화 옵션을 활성화하거나, `personal/` 을 동기화 제외 폴더로 설정 권장.
- Obsidian Sync와 이 starter kit의 Git 기반 동기화를 동시에 사용하면 충돌이 발생할 수 있음. 하나만 선택.
